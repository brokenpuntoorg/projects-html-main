Dominando el Foliador: Cómo Configurar y Usar un Numerador Automático

Aprende a ajustar la repetición y los dígitos de tu foliador automático de 6 o más números para un marcado rápido y secuencial de documentos.

El foliador (o numerador) automático es una herramienta indispensable para llevar un control secuencial en documentos, facturas o inventarios. Su mecanismo metálico de alta calidad garantiza un marcado preciso y duradero. Sigue esta guía paso a paso para configurarlo correctamente.

Paso 1: Preparación de la Tinta (Recarga)

La tinta es el corazón de tu foliador. **¡Atención!** Es crucial usar solo tinta especial para máquinas numeradoras. Las tintas comunes para sellos de caucho pueden dañar severamente el mecanismo de metal por su composición oleosa o ácida.

  1. Presiona el mango del foliador hasta la mitad del camino y utiliza la palanca de bloqueo (si tu modelo la tiene) para inmovilizarlo.
  2. Localiza y retira la almohadilla de tinta (generalmente presionando un broche o patillas de encaje).
  3. Entinta la almohadilla con la **cantidad justa** de tinta especial. Llenar demasiado o muy poco hará que los caracteres sean ilegibles.
  4. Vuelve a insertar la almohadilla en su guía y libera el mango del bloqueo.

Paso 2: Ajustar el Número Inicial

Antes de comenzar a sellar, debes establecer el número desde el que quieres empezar (por ejemplo, el 000001).

  1. Presiona el mango completamente hacia abajo y mueve la palanca de seguridad o bloqueo a la posición de *apagado* (OFF). Esto inmovilizará las ruedas de números y las dejará expuestas.
  2. Con el **palillo plástico o punzón** que se suministra con el foliador (nunca uses piezas metálicas), mueve las ruedas de números hasta obtener la cifra deseada.
  3. Si no requieres todos los dígitos (por ejemplo, solo 4 de 6), puedes hundir ligeramente los dígitos no deseados (los de la izquierda) en el cero con el punzón, para que queden por debajo del nivel de sellado.
  4. Una vez ajustado, desbloquea la palanca o presiona el mango para volver a la posición inicial.

Paso 3: Configurar la Secuencia de Repetición

Esta es la función clave del foliador automático. En la parte posterior del aparato, encontrarás un pequeño indicador de repetición o una palanca que se mueve entre ranuras numeradas (generalmente de 0 a 4).

Posición del Indicador Significado Uso Común
**0** Repite el mismo número indefinidamente (no avanza). Sellado de un lote con el mismo código.
**1** Avanza consecutivamente con cada sello (1, 2, 3...). Numeración de facturas o boletos.
**2** Repite el mismo número **2 veces** antes de avanzar (1, 1, 2, 2...). Original y copia.
**3 / 4 / etc.** Repite el mismo número la cantidad de veces indicada. Juegos de documentos.

Mueve la aguja indicadora a la ranura que corresponda a la secuencia que deseas utilizar (por ejemplo, a la posición **1** para numeración continua).

Paso 4: ¡A Foliar!

Una vez configurado, simplemente presiona el mango firmemente sobre el papel. La almohadilla girará y el mecanismo interno avanzará automáticamente según la repetición que hayas seleccionado.

Tip de Mantenimiento: Si el mecanismo se ensucia mucho, puedes limpiarlo con aceite, bencina o queroseno, aplicándolo con un pincel para asegurar un correcto cambio de número.
