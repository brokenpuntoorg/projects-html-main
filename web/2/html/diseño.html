Dominando el Foliador: Cómo Configurar y Usar un Numerador Automático

Aprende a ajustar la repetición y los dígitos de tu foliador automático de 6 o más números para un marcado rápido y secuencial de documentos.

El foliador o numerador automático es una herramienta esencial para la organización y archivo de grandes volúmenes de documentos, facturas o boletos. Su mecanismo metálico permite un marcaje rápido y secuencial. Aquí te explicamos cómo configurarlo y mantenerlo.

Paso 1: Conoce tu Numerador y Ajusta los Dígitos

Los foliadores se componen de ruedas metálicas que giran automáticamente. Para empezar a usarlo:

  1. Elige la Posición de Inicio: Usa el punzón de plástico que viene con tu foliador (o un objeto similar y delgado) para girar manualmente las ruedas hasta el número de inicio que desees (ej: 000001).
  2. Asegura el Mecanismo: Los foliadores suelen tener un mango que, al presionarlo hacia abajo, deja al descubierto las ruedas. Es en esta posición que puedes girarlas con el punzón.
  3. Bloquea los Dígitos Innecesarios: Si tu foliador tiene 8 dígitos pero solo necesitas 6, empuja las primeras dos ruedas (de izquierda a derecha) hacia el cuerpo del sello con el punzón. Esto las bloqueará para que no impriman.
Tip de Precisión: Asegúrate de que todas las ruedas estén bien alineadas antes de hacer la primera prueba de estampado para evitar impresiones borrosas o números a medio marcar.

Paso 2: Dominando el Dial de Repetición (Las Posiciones R0 a R6)

Esta es la parte más crucial. El foliador tiene un pequeño dial o anillo selector que controla cuándo debe aumentar el número secuencial.

Posición del Dial Acción de Numeración Uso Recomendado
**R0 o '0'** **No avanza.** Imprime siempre el mismo número. Para estampar números de lote o fecha fijos.
**R1 o '1'** **Avanza 1 dígito** con cada estampado. El uso más común: marcar documentos secuencialmente (1, 2, 3...).
**R2** **Avanza 1 dígito** después de 2 estampados. Para estampar boletos duplicados o archivos que requieran copia.
**R3 a R6** **Avanza 1 dígito** después de 3, 4, 5, o 6 estampados, respectivamente. Usos especializados que requieren múltiples copias con el mismo número.

Paso 3: Mantenimiento y Tinta

Los foliadores son máquinas de precisión. Para una larga vida:

  • **Tinta Esencial:** Usa **solo** tinta a base de aceite mineral, específica para numeradores. Esta tinta no solo entinta sino que también lubrica los engranajes internos, evitando la oxidación.
  • **Nunca uses:** Tinta base agua para sellos de goma comunes o tinta base solvente, ya que corroerán el metal.
  • **Limpieza:** Si el mecanismo se atasca, límpialo suavemente con un cepillo pequeño y un poco de diluyente de tinta de foliador (no alcohol).
¡PELIGRO! El mayor error es usar tinta común. El daño por oxidación no es reversible. ¡Asegúrate de recargar solo con tinta de foliador!
