SISTEMA DE ENCENDIDO Y TIEMPO MOTOR - LÍNEA SENSORES DE POSICIÓN DE ÁRBOL DE LEVAS (CMP)

SD22342 Ford / Plataforma Motores L4 Duratec 2.0L y 2.5L (08-15)

Referencias Técnicas

Tipo: Sensor de Posición de Árbol de Levas (CMP / Camshaft Position Sensor)

Tecnología: Sensor de Efecto Hall de 3 Terminales con Sincronización Avanzada

Configuración: Sonda de Inserción Escalonada Gruesa con Oreja de Sujeción Lateral Excéntrica

Sellado: Sistema de Doble Anillo O-Ring Negro para Máxima Hermeticidad contra Presión de Aceite

Conector: Terminal Rectangular Plano de 3 Pines Machos en Distribución Lineal Continua

Observaciones

Módulo de control de fase de árbol de levas CMP basado en tecnología de Efecto Hall de 3 vías, diseñado para la arquitectura moderna de motores L4 Duratec de 2.0L y 2.5L de Ford. A diferencia de las versiones inductivas básicas, el modelo SD22342 integra un circuito electrónico activo que requiere alimentación externa para generar una señal digital cuadrada de alta fidelidad. Físicamente cuenta con un cuerpo robusto que incorpora un sistema de doble junta tórica (O-ring) elastomérica para soportar las exigencias de presión de aceite en la cabeza del motor. Su funcionamiento es indispensable para que el módulo PCM gestione el sistema de distribución variable (VCT), optimice los pulsos de inyección y el avance de la chispa. El desgaste de este componente suele ocasionar códigos de falla OBD-II relativos al rango/rendimiento del circuito CMP, pérdida notable de potencia a altas revoluciones y un incremento drástico en las emisiones contaminantes.
