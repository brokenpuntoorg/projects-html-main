Observaciones

Este sensor de posición del cigüeñal (CKP) cuenta con una extensión de arnés integrada y está diseñado para motores compactos General Motors de 4 cilindros (L4), incluyendo las motorizaciones 1.4L y 2.0L utilizadas en plataformas emblemáticas como el Chevrolet Chevy y el Pontiac Sunbird. Al estar expuesto a vibraciones y rangos térmicos extremos cerca del bloque, su arnés remoto permite una conexión segura alejada de las zonas de mayor calor directo. El sensor se encarga de leer de forma precisa las revoluciones y la fase del motor para coordinar los pulsos de inyección y el salto de chispa. Las fallas comunes de este componente provocan que el motor se apague al alcanzar temperatura de trabajo, dificultades de arranque (marcha estéril), pérdida de potencia o jaloneos intermitentes.
